# 西南大学学位论文的 Typst 模板
本模板基于Typst 版本：0.13.0。

目前包含硕士、博士两种模板，基本完全实现了西南大学关于学位论文的要求。

本模板使用的命令和函数相对浅显易懂，并配有充分的注释。有一定typst基础的同学可以进行修改以满足自己的特定需求。

如果有问题请发邮件到chang1986@swu.edu.cn进行反馈。

如果感觉不错，请好评！
